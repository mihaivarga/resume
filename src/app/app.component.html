SUMMARY

Hello Reader,

Ever since my childhood I've been passionate about technology. Getting my first computer (with big sacrifices) from my parents was the best gift ever. I was already into some hardware / software home hacking even before getting the new personal gadget, in local town kids computer science clubs. My career path was starting to get clear each day.

Next years I studied computer science oriented high school, in the intensive computer science class. Even there I was into finding out more about new technologies, both software and hardware, to be able to create fun and helpful applications. Unfortunately, classes felt like old tech, however I learned hard and aced them.

Naturally my faculty years couldn't have been anything else than attending computer science university, UBB locally in my home town Cluj-Napoca (English courses). There while courses started to get complicated and somehow interesting, I still was eager to learn non-curriculum-oriented technologies, in those days, so I started working as web developer (PHP). Web development was not a course back then. When I was getting my degree, I had already switched several jobs at some companies while programming with different interesting technologies (PHP, JS, C#, C++, SQL, the list can go on). Programming languages felt like just another syntax to be learned, framework looked more like tools I can use, so all these were not a blocking step to developing and delivering projects.

The next years opportunity found me to start working as a freelancer, jumping from a tech stack to another, always looking for new ways to improve my experience and development performance. Learned how to manage teams. Learned now to take a project from client description to deployment. Learned how to find simple and active ways of engaging junior developers into the experience I had as a freelancer. Learned how to teach and share what I know to others through open source or junior developer guidance meetings.

More details about my freelance experience can be looked into below in my Experience section. Hopefully you like what you see.

With the hopes that reading this has not bored you a ton,

I thank you Reader.
